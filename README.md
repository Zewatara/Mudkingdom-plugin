# Mudkingdom Minecraft Plugin

---

This is a Minecraft plugin for my private Minecraft server. I started this project to learn a bit about Java.

## Building

To build this you need:

1. [Maven](https://maven.apache.org/download.cgi)
2. Run the following command:
```
   mvn clean package
```
3. Check the `target/` folder for the created jar

## Current Features

- ✓ Plugin file setup
- ✗ Area detection
